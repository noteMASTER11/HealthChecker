# Third-Party Notices

## WinMTR reference

HealthChecker includes a traceroute/statistics workflow inspired by WinMTR behavior
(e.g., per-hop sent/received/loss/best/avg/worst/last metrics).

The WinMTR source tree may be used locally as an implementation reference during
development, but it is not required to build or run this project.

Project reference:
- WinMTR: https://github.com/White-Tiger/WinMTR

If you redistribute WinMTR code/assets directly, you must include the original
license and attribution according to that project terms.
